# PMI_IA: Inteligencia Artificial para la Dirección de Proyectos (PMBOK® 8ª Edición, 2025)

Este proyecto es un entorno de trabajo y aprendizaje diseñado para asistir a directores de proyectos y agentes de Inteligencia Artificial (LLMs) en la gestión de iniciativas bajo los estándares del **PMBOK® 8ª Edición (2025)**, la **Guía Scrum 2020** y marcos de trabajo ágiles/híbridos. 

El repositorio contiene guías de estudio estructuradas, plantillas de entregables, ejemplos de implementación práctica, recursos bibliográficos de referencia y archivos multimedia.

## 🛠️ ¿Cómo usar este espacio?

### Para Directores de Proyecto (Humanos)
1.  **Estudio y Capacitación**: Consulta la [00_GuiaDeEstudio.md](00_GuiaDeEstudio.md) para comprender rápidamente la transición hacia el PMBOK® 8.
2.  **Gestión Práctica**: Copia el contenido de la carpeta [Entregables/](Entregables) a un nuevo directorio para tu proyecto y complétalo siguiendo las instrucciones entre corchetes.
3.  **Ejemplo**: Revisa [Entregables ejemplo - proyecto A/](Entregables%20ejemplo%20-%20proyecto%20A) para ver cómo debe lucir un set de artefactos terminado.

### Para Agentes de Inteligencia Artificial (LLMs)
Este repositorio está optimizado para su consumo por agentes conversacionales de IA:
1.  **Instrucción de Contexto**: Al iniciar la sesión, lee el archivo [AGENTS.md](AGENTS.md) para adoptar el rol, los principios rectores y la estructura de respuestas.
2.  **Búsqueda Normativa**: Utiliza los archivos en [Recursos/](Recursos) o las guías de estudio para justificar teóricamente cualquier recomendación dada al usuario basándote en la 8ª edición.
